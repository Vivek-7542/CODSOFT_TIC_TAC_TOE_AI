# 🎮 Tic-Tac-Toe AI

An intelligent Tic-Tac-Toe game built using Python, where the player competes against an AI opponent. The AI uses the Minimax Algorithm to make optimal moves and provide a challenging gameplay experience.

## 📌 Features

- Play Tic-Tac-Toe against an AI opponent
- AI powered by the Minimax Algorithm
- Unbeatable gameplay strategy
- Interactive command-line interface
- Easy-to-understand Python implementation

## 🛠️ Technologies Used

## 🚀 Getting Started

### Prerequisites

Make sure Python 3 is installed on your system.

Check your Python version:

```bash
python --version
```

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe-ai.git
```

2. Navigate to the project directory:

```bash
cd tic-tac-toe-ai
```

3. Run the game:

```bash
python tic_tac_toe.py
```

## 🎯 How to Play

- The game board consists of 9 cells numbered from 1 to 9.
- Enter the position number where you want to place your mark.
- You play as **X** and the AI plays as **O**.
- The AI will automatically make its move after yours.
- The player who gets three marks in a row, column, or diagonal wins.

Example Board:

```
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```

## 🧠 AI Algorithm

This project uses the **Minimax Algorithm**, a decision-making algorithm commonly used in game theory and artificial intelligence.

### How Minimax Works

- Evaluates all possible game states.
- Maximizes the AI's chances of winning.
- Minimizes the player's chances of winning.
- Ensures optimal moves at every turn.
